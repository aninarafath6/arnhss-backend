# arnhss-backend

## Installation Setup

Clone this repo
```
git clone git@github.com:muhammadsemeer/arnhss-backend.git
```

## System Requirements

```
Node v16.17.1
npm 8.15.0
mysql Ver 8.0.30
```


Follow these steps
```
npm install

npm run install:hooks # optional - it will run automatically on postinstall

npm run dev # start development server

```
